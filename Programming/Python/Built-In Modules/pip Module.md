```python
# Help page of pip
pip help

# Help page of a specific function
pip help install

# List all Python Packages installed
pip list

# Get info about a Python Package (Should be installed)
pip show urllib3

# Install a package only for the specific user
pip install --user [package]

# Update a package
pip install -U [package]

# Install a specific version of a package
pip install pygame==1.9.2
```
