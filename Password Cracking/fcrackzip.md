```sh
fcrackzip -v -D -u -p /usr/share/wordlists/rockyou.txt save.zip
```

*-v : Enable Verbosity
-D : Use A Dictionary Attack
-u : Use Unzip To Weed Out Wrong Passwords
-p : use string as initial password/file*

<u>/usr/share/wordlists/rockyou.txt.gz</u> : Wordlist
<u>save.zip</u> : File To Be Cracked
